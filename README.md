# hello
A hello word cordova + pixi mobile app

## Requirements

- node
- yarn
- cordova

## Installation

```
yarn install
yarn create
```
 
## Test in the browser

```
yarn start
```

## Generate dev bundles and copy to Cordova subfolder

```
yarn build
```

## Test with the device plugged

```
yarn debug
```

## Generate a minified / optimized build

```
yarn release
```
